# ECE440Proj
DDMR controlled using Raspberry Pi 5

## Hardware
- Raspberry pi 5
- 3 Ultrasonic sensors
- 2 L298N motor drivers
- Wires
- Resistors

## Power Considerations
- battery pack for motor driving
- mobile power supply 5V/5A for RPi5

## Motor Driver (L298N)
### Node 1 (Front)
- Iin1 = pin 17 on RPi
- Iin2 = pin 22 on RPi
- Iin3 = pin 23 on RPi
- Iin4 = pin 24 on Rpi
